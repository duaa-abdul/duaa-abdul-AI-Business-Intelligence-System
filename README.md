# AI-Driven Business Intelligence System

## Project Overview
This project analyzes the Superstore dataset using machine learning and data visualization techniques to generate business insights and predictions.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis
- Machine Learning models (XGBoost, ARIMA, KMeans)
- Interactive Streamlit Dashboard
- Business insights and recommendations

## Tools & Libraries
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Plotly
- Streamlit

## How to Run

### 1. Install requirements
pip install streamlit pandas numpy plotly scikit-learn xgboost

### 2. Run notebook
Open `Task1.ipynb`

### 3. Run dashboard
streamlit run app.py

## Output
- KPI Dashboard
- Forecasting graphs
- Customer segmentation
- Risk analysis heatmap
